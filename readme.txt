This plugin allows you to display your latest Medium articles anywhere on your blog with the use of a simple shortcode.

== Installation ==

1. Upload the `medium parser` folder to the `/wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
